# ducational purposes only. it is not meant to break any policies and must be used after reading the TOS of chess.com and never to be used in competitive matches Project Chess Bot (v3.0 This is a Chrome extension that integrates Stockfish for analysis on Chess.com.

### (disclaimer) Auto play doesn't work
## What's New in v3.0
Improved Reliability: Added FEN caching and API retry with exponential backoff for better stability.
Enhanced Humanization: Smarter move delays based on complexity and varied click patterns to avoid detection.
Settings Persistence: Auto-play and pause states now persist across sessions.
UI Improvements: Real-time engine status indicator (Cloud/Local) and pause/resume button.
Board Verification: One-click board check panel shows current FEN, best move, and evaluation without leaving the page.
Tabbed Interface: Organized into Bot and Board tabs for better navigation.
Live Board View: Dedicated tab showing the current board state with FEN and turn information.
Animated Background: Centipede-like particle animation that follows the mouse cursor.
Memory Optimization: Limited game history to prevent memory leaks in long games.
New Icon: Added a custom chess-themed icon for the extension.

## How to Install
Download and Unzip: Download the .zip file and extract it to a folder.
Open Extensions Page: In Google Chrome, go to chrome://extensions/.
Enable Developer Mode: Toggle the switch in the top-right corner.
Load Unpacked: Click the "Load unpacked" button in the top-left.
Select Folder: Select the folder containing these files (where manifest.json is located).
